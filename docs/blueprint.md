# **App Name**: KR Parking Pro

## Core Features:

- Product Parking Limits: Define and manage maximum parking slots for each product category (LPG, Propylene, etc.).
- Entry Slip Generation: Automatically generate a unique token number and entry slip upon truck arrival, capturing key details such as date, truck number, product type, time in, and the assigned token number.
- Exit Slip Generation: Upon truck departure, automatically generate an exit slip containing date, truck number, product, time in, time out, token number, and parking duration, ensuring accurate record-keeping and billing.
- Live Dashboard: Display real-time parking data, including available slots per product, truck details, and overstayed vehicle alerts.
- Reporting: Generate reports, filter product-wise truck details, generate monthly and yearly summaries, and send alerts for overstayed vehicles.
- PDF/Excel Export: Export all reports into PDF or Excel format for documentation and offline analysis.
- Parking Prediction: Leverage historical data using AI as a tool to forecast future parking demand by product type. The prediction will be shown to admin users.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) to evoke a sense of reliability and authority in an industrial setting.
- Background color: Light gray (#E5E8E8), offering a neutral backdrop that ensures legibility and reduces visual strain.
- Accent color: Teal (#2E86AB) for key interactive elements to draw the user's attention.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and easily readable text.
- Code font: 'Source Code Pro' for any config files shown on screen.
- Use consistent and clear icons to represent different product categories, truck status, and report types.
- Employ subtle animations (e.g., loading indicators, smooth transitions) to enhance the user experience and provide feedback during interactions.
